# ReactApp
This project is about React, NodeJS and MongoDB.

## Getting Started


### 1. Install MongoDB via home brew on mac
```
brew install mongodb
```

### 2. Go to server path inside React folder
```
cd server
```

#### 2.1 Install Express, body-parser, cors and mongoose in backend file
```
npm install express body-parser cors mongoose
```

- Start MongoDB
```
Mongod
```

- Open new terminal tap to access database
```
Mongo
```

- Create / call database
```
use customers
```

#### 2.2 Install Nodemon
```
npm install -g nodemon
```

- Start Node server
```
nodemon server
```

### 3. Go to reactApp path inside React folder
```
cd test-app
```

- Start React app
```
npm start 
```
